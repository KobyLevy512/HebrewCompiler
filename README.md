# שפת תכנות בעברית
התוכנה מתרגמת מעברית לתוכנית מוגמרת בעזרת טכנולוגיית
.Net

משמע הדבר כל הדבר הקיים בשפת סי שארפ קיים גם במתרגם.
השפה עצמה היא שפת מונחה-עצמים, הגרסא הראשונה יצאה בדצמבר 2021.
